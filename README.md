# Echo Orbit - 4人対人リアルタイムステルスゲーム

暗闇の中で音波を使って周囲を一瞬だけ可視化し、足音や残響で敵を探るリアルタイムマルチプレイゲーム。4人自動マッチメイキング対応。

## 操作方法

- `WASD` / `矢印キー`：移動
- `Space`：音波発射
- `R`：再開 / リスタート
- 「マッチメイキング開始」ボタン：サーバーに接続して4人マッチング

## 仕様

- **暗闇ステルス**：画面は常に真っ暗。音波でしか見えない
- **音波スキャン**：音波を発射して周囲の壁と敵を一瞬だけ可視化
- **足音検知**：移動時の足音で敵が位置を察知する
- **敵AI**：音波や足音を聞いて自動追跡、壁があれば遠回りして追いかけてくる
- **対人同期**：最大4人が同じルームに自動マッチング
- **リアルタイム同期**：プレイヤー位置・音波・足音が同期される

## アーキテクチャ

### フロントエンド
- **ホスティング**：Cloudflare Pages
- **技術**：HTML5 Canvas, WebSocket, Vanilla JavaScript

### バックエンド
- **ホスティング**：Railway
- **技術**：Node.js + Express + WebSocket
- **機能**：マッチメイキング、ルーム管理、プレイヤー同期

## デプロイ方法

### バックエンド（Railway）

1. Railway にプロジェクトを作成
2. GitHub リポジトリを接続
3. Start Command：`npm start`

### フロントエンド（Cloudflare Pages）

1. Cloudflare Pages で新規プロジェクトを作成
2. GitHub リポジトリを接続
3. ビルド：直接ホスティング（`index.html` をルートに）

## ゲームの流れ

1. 「マッチメイキング開始」をクリック
2. Railway サーバーに接続
3. 4人揃うまで待機
4. 自動的にルーム作成＆ゲーム開始
5. 4人でサバイバル

## 技術仕様

- **マッチメイキング**：4人揃うまでキュー管理
- **ルーム管理**：複数ルームを同時運用
- **状態同期**：125ms ごとに送信
- **WebSocket**：リアルタイムプレイヤー同期
