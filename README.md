# Echo Orbit

暗闇の中で音波を使って周囲を一瞬だけ可視化し、足音や残響で敵を探るリアルタイムステルスプロトタイプです。

## 操作方法

- `WASD` / `矢印キー`: 移動
- `Space`: 音波発射
- `R`: 再開 / リスタート
- ルーム名を入力して「接続」: オンライン同期モード

## 仕様

- 画面は常に暗闇
- 音波を発射すると周囲の壁や敵が輪郭として一瞬表示される
- 音波は敵に位置を知らせる
- 足音や残響で周囲の敵の存在を察知する

## 実行方法

`index.html` をブラウザで開いてください。
